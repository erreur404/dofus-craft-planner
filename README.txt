# Installation

checkout the code or download as zip

install python from https://www.python.org/downloads/ (if you have't already ;) )

cd into the downloaded folder

run "pip install -r requirements.txt" to install the dependencies

run using "python main.py"

A Google chrome window should open with the right url.
If not, connect to the local server with a browser (default : http://localhost:5000)
